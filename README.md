# Descripción del proyecto

Este proyecto desarrolla una aplicación en Python capaz de:

- Leer archivos DICOM
- Extraer metadatos clínicos
- Procesar imágenes médicas usando OpenCV
- Generar imágenes procesadas
- Exportar información a CSV usando Pandas

El objetivo es simular parte del flujo de trabajo de un sistema PACS utilizado en informática médica.

# Librerías utilizadas

- pydicom
- numpy
- pandas
- opencv-python
- matplotlib
